# CryptoLib
An educational library with different hash and cryptographic functions

## Contents

#### Hash functions

1. MD5
2. SHA1
3. SHA2

#### Encoding

1. Base64 


#### Cryptographic functions

1. ROT
2. DES 
3. RSA (1024 bits)

#### Cryptographic attacks

1. MITM for DES (with 56 known bits)
2. Wiener's Attack for RSA
